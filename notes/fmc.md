# FMCとは

fusion_place マネジメントコンソール（FMC）は、fusion_place cloud の運用状況を確認・管理するための専用コンソールです。  

---

## ■ 主な機能

- **アップデート**  
  アップデート情報の確認および実施日程の変更ができます。

- **サービスステータス**  
  サービス稼働状況の確認、サービス再起動ができます。

- **サーバーログ**  
  サーバー側ログ（直近14日分）の確認ができます。

- **メトリクス**  
  アラート通知基準で使用しているメトリック（直近14日分）の確認ができます。

- **バックアップ**  
  データベースのバックアップ状況を確認できます。

- **サマリレポート**  
  メトリクス（日次ベース）の確認ができます。

- **監査ログ**  
  fusion_place cloud の監査ログを確認できます。

- **IPアドレスの登録・削除**  
  セキュリティ制限として登録した IP アドレスの管理ができます。
